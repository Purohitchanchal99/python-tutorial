Python Functions and Arguments

Python Functions are blocks of statements that perform specific tasks. They allow you to group commonly or repeatedly performed tasks together and make them reusable. Instead of writing the same code repeatedly for different inputs, you can call functions to execute the code contained within them.

Benefits of Using Functions

  • Enhance Code Readability
  • Promote Code Reusability

Python Function Declaration

The syntax to declare a function is:

def function_name(parameters): """Docstring""" # body of the function return expression

Types of Functions in Python

There are mainly two types of functions in Python:

  • Built-in library functions: These are standard functions in Python that are available for use.
  • User-defined functions: You can create your own functions based on your requirements.

Creating a Function in Python

You can create a user-defined function in Python using the def keyword. You can add any functionality and properties to it as needed:

# A simple Python function def fun(): print("Welcome to GFG")

Calling a Python Function

After creating a function in Python, you can call it by using the function name followed by parentheses containing parameters (if any) for that function:

# A simple Python function def fun(): print("Welcome to GFG") # Driver code to call the function fun()

Python Function with Parameters

You can define and call functions with parameters. Parameters are used to pass values to functions. You can specify parameter data types and return types:

def add(num1: int, num2: int) -> int: """Add two numbers""" num3 = num1 + num2 return num3 # Driver code num1, num2 = 5, 15 ans = add(num1, num2) print(f"The addition of {num1} and {num2} results {ans}.")

Python Function Arguments

Arguments are the values passed inside the parentheses of a function. A function can have any number of arguments, separated by commas. For example, here's a function that checks whether a number is even or odd:

# A simple Python function to check whether x is even or odd def evenOdd(x): if (x % 2 == 0): print("even") else: print("odd") # Driver code to call the function evenOdd(2) evenOdd(3)

Types of Python Function Arguments

Python supports various types of function arguments:

  • Default arguments: Parameters with default values.
  • Keyword arguments (named arguments): Arguments with specific parameter names.
  • Positional arguments: Arguments passed based on their positions.
  • Arbitrary arguments (variable-length arguments *args and **kwargs): Arguments that can vary in number and keyword arguments.
