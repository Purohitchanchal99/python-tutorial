Python Input and Output

Python Output

In Python, we can use the print() function to display output. For example:

        
print('Python is powerful')

# Output: Python is powerful

The print() function displays the string enclosed inside single quotations.

Syntax of print()

The print function accepts several parameters:

  • object: Value(s) to be printed.
  • sep (optional): Allows separation of multiple objects inside print().
  • end (optional): Allows the addition of specific values like a new line "\n" or a tab "\t".
  • file (optional): Specifies where the values are printed. Default value is sys.stdout (screen).
  • flush (optional): A boolean specifying if the output is flushed or buffered. Default: False.

Example: Python Print Statement

        
print('Good Morning!')
print('It is rainy today')

Output:

        
Good Morning!
It is rainy today

Example: Python print() with end Parameter

        
# Print with end whitespace
print('Good Morning!', end=' ')

print('It is rainy today')

Example: Python print() with sep Parameter

        
print('New Year', 2023, 'See you soon!', sep='. ')

Example: Print Python Variables and Literals

        
number = -10.6
name = "Programiz"

# Print literals     
print(5)

# Print variables
print(number)
print(name)

Example: Print Concatenated Strings

        
print('Programiz is ' + 'awesome.')

Output Formatting

Sometimes we would like to format our output to make it look attractive. This can be done by using the str.format() method. For example:

        
x = 5
y = 10

print('The value of x is {} and y is {}'.format(x, y))

Python Input

When programming, we often want to take input from the user. In Python, we can use the input() function.

Syntax of input()

        
input(prompt)

Here, prompt is the string to display on the screen (optional).

Example: Python User Input

        
# Using input() to take user input
num = input('Enter a number: ')

print('You Entered:', num)

print('Data type of num:', type(num))

Output:

        
Enter a number: 10
You Entered: 10
Data type of num: <class 'str'>

It's important to note that the entered value "10" is a string, not a number. To convert user input into a number, we can use int() or float() functions, like this:

        
num = int(input('Enter a number: '))

Here, the data type of the user input is converted from a string to an integer.
