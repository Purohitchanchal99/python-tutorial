Python Class and Object

In Python, a class is a blueprint for creating objects. An object is an instance of a class, which encapsulates data (attributes) and behavior (methods). The class defines the structure and behavior, while the object is an actual instance of that structure.

Define a Class

        
class Car:
    brand = ""
    model = ""
    
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model
    
    def display_info(self):
        print(f"{self.brand} {self.model}")

Create an Object

        
# create an object of the Car class
car1 = Car("Toyota", "Camry")

# access attributes and call methods
car1.display_info()

Output: Toyota Camry

Object Methods

Object methods are functions defined within a class that operate on the attributes of an object. They represent the behavior associated with the class. In the example below, the display_info method is an object method. 

        
class Car:
    brand = ""
    model = ""
    
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model
    
    def display_info(self):
        print(f"{self.brand} {self.model}")

# create an object of the Car class
car1 = Car("Toyota", "Camry")

# access attributes and call methods
car1.display_info()

Output: Toyota Camry

Example

Let's create a class called Car with attributes brand and model. We'll define a constructor (__init__) to initialize the object's attributes and an object method (display_info) to display information about the car.

In the example below, we'll create an object (car1) of the Car class, set its attributes, and call the display_info method to print the car's information. 

        
# create an object of the Car class
car1 = Car("Toyota", "Camry")

# access attributes and call methods
car1.display_info()

Output: Toyota Camry
