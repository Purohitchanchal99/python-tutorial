Python Functions Guide

Functions in Python are blocks of statements that perform a specific task. They enhance code readability and reusability by grouping common tasks together. Here are some benefits of using functions:

Benefits of Using Functions

Functions in Python offer several advantages, including:

1. Increase Code Readability

2. Increase Code Reusability

Python Function Declaration

The syntax for declaring a function in Python is:


def function_name(parameters):
      """Docstring"""
      # body of the function
      return expression

Types of Functions in Python

There are mainly two types of functions in Python:

1. Built-in library function: Standard functions in Python that are available to use.

2. User-defined function: Functions created based on specific requirements.

Creating a Function in Python

We can create a user-defined function in Python using the def keyword. Here's a simple example:


# A simple Python function 
def fun():
    print("Welcome to GFG")
fun()

Python Function with Parameters

If you have experience in C/C++ or Java, you can define the data type of arguments and the return type in Python (specifically for Python 3.5 and above). Example:


def add(num1: int, num2: int) -> int:
    """Add two numbers"""
    num3 = num1 + num2
    return num3

num1, num2 = 5, 15
ans = add(num1, num2)
print(f"The addition of {num1} and {num2} results {ans}.")

Types of Python Function Arguments

Python supports various types of function arguments:

Docstring

The first string after the function is called the Docstring. It is used to describe the functionality of the function. Example:


# A simple Python function to check whether x is even or odd
def evenOdd(x):
    """Function to check if the number is even or odd"""
    if (x % 2 == 0):
        print("even")
    else:
        print("odd")
print(evenOdd.__doc__)

Python Function within Functions

A function defined inside another function is known as the inner function or nested function. Nested functions can access variables of the enclosing scope. Example:


# Python program to demonstrate accessing variables of nested functions
def f1():
    s = 'I love Green'
    def f2():
        print(s)
    f2()
f1()

Anonymous Functions in Python

Anonymous functions in Python, known as lambda functions, are functions without a name. Example:


# Python code to illustrate the cube of a number using a lambda function
cube = lambda x: x*x*x
print(cube(7))

Return Statement in Python Function

The return statement is used to exit from a function and return a specified value to the caller. Example:


def square_value(num):
    """This function returns the square value of the entered number"""
    return num**2
print(square_value(2))

Pass by Reference and Pass by Value

Parameter passing in Python is similar to reference passing in Java. Example:


# Here x is a new reference to the same list lst
def myFun(x):
    x[0] = 20

# Driver Code (Note that lst is modified after function call)
lst = [10, 11, 12, 13, 14, 15]
myFun(lst)
print(lst)

When we pass a reference and change the received reference to something else, the connection between the passed and received parameter is broken. For example:


def myFun(x):
    x = [20, 30, 40]

# Driver Code (Note that lst is not modified after function call)
lst = [10, 11, 12, 13, 14, 15]
myFun(lst)
print(lst)

Another example demonstrates that the reference link is broken if we assign a new value (inside the function):


def myFun(x):
    x = 20

# Driver Code (Note that x is not modified after function call)
x = 10
myFun(x)
print(x)
