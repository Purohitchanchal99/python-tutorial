Python Break and Continue

In Python, the break and continue statements are used to control the flow of loops.

The break statement is used to exit the current loop prematurely. It is often used to terminate a loop when a certain condition is met.

for i in range(1, 6):

 if i == 3:

  break

 print(i)

The above code will print numbers from 1 to 2 because the loop is terminated when i == 3.

The continue statement is used to skip the current iteration of a loop and continue with the next iteration. It is often used when you want to skip certain items in a loop.

for i in range(1, 6):

 if i == 3:

  continue

 print(i)

The above code will print numbers from 1 to 5, skipping the iteration where i == 3.
