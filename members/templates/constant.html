In Python, constants are usually declared and assigned in a module (a new file containing variables, functions, etc which is imported to the main file).

Let's see how we declare constants in a separate file and use them in the main file.

Create a constants.py file:

# Declare constants
PI = 3.14
GRAVITY = 9.8

Create a main.py file:

# Import the constant file we created above
import constants

print(constants.PI)  # Prints 3.14
print(constants.GRAVITY)  # Prints 9.8
