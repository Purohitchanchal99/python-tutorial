Control structures are an essential part of programming, allowing developers to dictate the flow and behavior of a program based on certain conditions or requirements. There are primarily three types of control structures:

Sequential Control:

In sequential control, statements are executed in a linear fashion, one after the other. The program follows a top-down approach, executing each statement in the order they are written.

Selection Control:

Selection control structures are used to make decisions in a program. They determine which section of code to execute based on conditions or criteria. The most common selection control structure is the "if" statement, which allows the program to take different paths depending on the truth or falsity of a condition.

Repetition Control:

Repetition control structures, also known as loops, are used to execute a section of code multiple times. Loops can be executed a fixed number of times or until a specific condition is met. Common loop types include "for" loops, "while" loops, and "do-while" loops.

Python if statement

The if statement is the most simple decision-making statement. It is used to decide whether a certain statement or block of statements will be executed or not.

Elif

The elif keyword is Python's way of saying "if the previous conditions were not true, then try this condition."

Example 1: Using Elif

      a = 33
      b = 33
      {% if b > a %}
        {{ "b is greater than a" }}
      {% elif a == b %}
        {{ "a and b are equal" }}
      {% endif %}

Else

The else keyword catches anything that isn't caught by the preceding conditions.

Example 2: Using Else

      a = 200
      b = 33
      {% if b > a %}
        {{ "b is greater than a" }}
      {% elif a == b %}
        {{ "a and b are equal" }}
      {% else %}
        {{ "a is greater than b" }}
      {% endif %}
